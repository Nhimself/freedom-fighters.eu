Follow the steps below to join the fun!

  1. Install ARMA Reforger: Buy from the official website or get it from the Steam Store.
  2. Start the Game: Launch ARMA Reforger and select "Multiplayer."
  3. Find the Server: Search for freedom-fighters.eu/Reforger in the server browser.
  4. Select the Server: Highlight it in the list on the left.
  5. Enter the Password: Use the password provided on Discord.
  6. Download Required Mods: The game will automatically download and install mods. For details, visit the Mods Overview page.
  7. Restart (if Needed): If prompted, restart the game after downloading mods.
  8. Join the Mission: Load into the basic training mission on the server.
  9. Set Up Voice Chat: Adjust your in-game sound settings and bind a key for voice communication.
